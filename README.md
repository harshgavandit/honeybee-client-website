# HoneyBee Website

This repository contains the HoneyBee Website frontend application built with Next.js, TypeScript, Tailwind CSS, and React.

## Project structure

- `Frontend/` — Next.js frontend app
  - `src/app/` — application routes and pages
  - `src/components/` — reusable UI components
  - `src/lib/` — data and utility modules

## Requirements

- Node.js 18+ (or compatible version)
- npm

## Setup

```bash
cd Frontend
npm install
```

## Run locally

```bash
cd Frontend
npm run dev
```

Then open `http://localhost:3000` in your browser.

## Build

```bash
cd Frontend
npm run build
```

## Lint

```bash
cd Frontend
npm run lint
```

## Notes

The frontend uses `eslint-config-next` and Tailwind CSS. If you change dependencies, run `npm install` again inside `Frontend/`.
